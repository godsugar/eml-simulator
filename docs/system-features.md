# システム概要 — 機能

## FR-1: 式木ビルダー

空スロット `_` にホバーするとメニューが表示され、3択から選ぶ:

| 選択肢 | 意味 |
|--------|------|
| `1` | 定数リーフ |
| `x` | 変数リーフ（入力した複素数 x が代入される） |
| `eml(_, _)` | EMLノード（子スロット2つが展開される） |

- 任意の深さまで入れ子にできる
- ノード右端の `×` でスロットに戻せる
- 式全体の括弧表記をリアルタイム表示
- 論文の指標「リーフ数 K」と「深さ」をリアルタイム表示

---

## FR-2: プリセット式

26種のプリセット式をワンクリックでロード。式木が即座に構築される。

| カテゴリ | プリセット |
|---------|-----------|
| 定数 | −1, 0, 1, 2, e, i, π |
| 指数・対数 | exp(x), ln(x) |
| 代数 | x, x+x, x\*x, x³, √x, 1/x |
| 三角関数 | sin(x), cos(x), tan(x) |
| 逆三角関数 | arcsin(x), arccos(x), arctan(x) |
| 双曲線関数 | sinh(x), cosh(x), tanh(x) |
| その他 | 空のスロット（手動構築用） |

---

## FR-3: 変数 x の入力

3つの入力モードを用意:

1. **数値入力** — 実部・虚部を個別に入力（任意の複素数）
2. **クイック選択** — 0, 1, −1, i, −i, e, π, 2, 1/2 のボタン
3. **スライダー** — 実部・虚部それぞれ −5 ～ 5 の範囲をスムーズにスイープ

---

## FR-4: 評価・ステップ表示

式木の各ノードで中間計算値をインライン表示:

- `exp(左辺)` の値（オレンジ色）
- `ln(右辺)` の値（ティール色）
- `ln(0)` は未定義として赤字で通知

ステップパネルでは評価トレースをインデント付きツリーで表示。

---

## FR-5: 複素平面プロット

- 現在の評価結果を複素平面（Re 軸 × Im 軸）にプロット
- スライダーで x を動かすと軌跡が描画される（最大200点）

---

## 追加機能（実装済み）

### 式木拡大モーダル

式木ビルダーの「⛶ 拡大」ボタンで全画面モーダルを開く。左右2ペイン構成:

- **左ペイン**: 式木（スクロール可）— コンパクトモードにも対応
- **右ペイン**: 幅320pxの関数グラフパネル（式の変更に追従してリアルタイム更新）

### 関数グラフ

実軸上で x を走査したときの `Re(f(x))` と `Im(f(x))` をラインチャートで表示。x の範囲を調整可能。

### 関数同定

組み立てた EML 式が既知のどの初等関数に相当するかを自動判定:

- PRESETS（「その他」カテゴリを除く全プリセット）を候補として実軸10点の数値サンプルで照合
- 最大誤差をスコア化してランキング表示
- 誤差 1e-6 以下は「一致」として緑バナーで表示
- プリセットを追加すると自動的に照合候補も増える

### 日英切り替え

ヘッダー右上のトグルボタン（`JA` / `EN`）で UI 言語を即時切り替え:

- 日本語（デフォルト）と英語に対応
- ボタンを押すだけで全 UI テキストが切り替わる
- アプリタイトル・ラベル・カテゴリ名・ヒント文など全文字列が対象
